## 1. Introduction (研究背景与意义)

### 1.1 问题背景
- 无人机(UAV)可以在空中自主组装，以应对不同的场景和任务需求
- **核心挑战**: 旋翼无人机空中自主对接问题研究较少，现有研究多集中在与悬停无人机的对接
- **本文目标**: 解决与**悬停或移动四旋翼**的对接问题

### 1.2 现有研究局限
- Reference [16]: 通过视觉和倾斜控制律实现旋翼无人机空中垂直组合
- Reference [17]: 提出ModQuad UAV概念，使用梯度场函数和速度控制完成与悬停无人机的对接
- Reference [18]: 基于ModQuad，使用视觉和加速度计数据融合作为导航信息
- **共同问题**: 方法相对简单，**缺乏抗干扰能力**

### 1.3 本文贡献
- 提出完整的四旋翼无人机空中自主对接技术方案
- 将对接过程分为三个阶段：**接近阶段**、**追踪阶段**、**对接阶段**
- 设计能够**自适应估计目标点速度和外部干扰**的对接控制器
- 仿真验证在干扰条件下与移动四旋翼的自主对接能力

## 2. Method (方法详解)

### 2.1 系统架构与坐标系定义

#### 五个坐标系:
1. **惯性坐标系 {I}**: 世界坐标系
2. **主动对接无人机机体坐标系 {B1}**: 与主动UAV固连，原点在质心
3. **被动对接无人机机体坐标系 {B2}**: 与被动UAV固连
4. **主动对接无人机相机坐标系 {C}**: 原点在相机光心，z轴与主光轴平行
5. **对接点坐标系 {D}**: 用于描述对接位置

#### 关键数学关系:
- **旋转矩阵**: 从惯性系到对接点坐标系的旋转矩阵 R_DI
- **位置计算**: 通过相机测量的标记中心位置 dC 和已知的偏航角 ψ，计算机体坐标系下的位置
- **对接点位置**: dB_a = f(dC, ψ, 几何关系)

### 2.2 三阶段对接策略

```
┌─────────────────────────────────────────────────────────┐
│                    对接过程三阶段                         │
├─────────────┬─────────────────┬─────────────────────────┤
│  接近阶段    │    追踪阶段      │       对接阶段          │
├─────────────┼─────────────────┼─────────────────────────┤
│ 远距离接近   │  中距离跟踪      │    近距离精确对接        │
│ 粗定位      │   精确定位       │    物理连接             │
└─────────────┴─────────────────┴─────────────────────────┘
```

#### 阶段1: 接近阶段 (Approach Phase)
- **目标**: 从远距离接近到对接目标附近
- **方法**: 基于粗略位置信息的导航
- **特点**: 允许较大误差，快速接近

#### 阶段2: 追踪阶段 (Tracking Phase)
- **目标**: 稳定跟踪移动的目标无人机
- **制导方法**: 根据阶段目标设计制导律
  - 计算期望速度方向
  - 保持安全跟踪距离
  - 实时更新目标位置

#### 阶段3: 对接阶段 (Docking Phase)
- **目标**: 完成物理对接
- **制导方法**: 精确位置控制
  - 减小相对速度
  - 精确对准对接机构
  - 完成机械连接

### 2.3 核心控制器设计

#### 自适应反步控制器 (Backstepping Adaptive Control)

**控制器结构:**
```
┌────────────────────────────────────────────────────────┐
│                    控制器架构                            │
├────────────────────────────────────────────────────────┤
│  输入层                                                 │
│  ├── 视觉测量: 标记位置 dC                              │
│  ├── 姿态信息: 偏航角 ψ                                 │
│  └── 传感器: 加速度计数据                                │
├────────────────────────────────────────────────────────┤
│  估计层 (自适应律)                                       │
│  ├── 目标速度估计: v̂_target                             │
│  └── 外部干扰估计: d̂                                    │
├────────────────────────────────────────────────────────┤
│  控制层 (反步控制)                                       │
│  ├── 位置误差 → 速度指令                                │
│  ├── 速度误差 → 加速度指令                              │
│  └── 加速度 → 控制力/力矩                               │
├────────────────────────────────────────────────────────┤
│  输出层                                                 │
│  └── 电机控制信号 (4个旋翼转速)                          │
└────────────────────────────────────────────────────────┘
```

#### 关键算法步骤:

**Step 1: 位置和姿态计算**
```python
# 伪代码
R_DI = Rz(ψ/2) * Rx(π/2) * Ry(ψ/2)  # 旋转矩阵
dB1 = R_B1C * dC + (R_I)^T * R_I * R_B2 * dG  # 位置计算
```

**Step 2: 自适应估计**
- 使用**自适应律**实时估计:
  - 目标无人机的速度 (v_target)
  - 外部干扰 (风扰等) (d)
- 估计误差通过Lyapunov函数保证收敛

**Step 3: 反步控制律设计**
- **外环**: 位置控制 → 产生期望速度
- **内环**: 速度控制 → 产生期望加速度
- **执行层**: 加速度 → 电机转速

**控制律公式:**
```
u = -k1·e_pos - k2·e_vel + v̂_target + d̂ + α

其中:
- u: 控制输入
- e_pos: 位置误差
- e_vel: 速度误差  
- v̂_target: 估计的目标速度
- d̂: 估计的干扰
- α: 前馈补偿项
```

### 2.4 制导方法 (Guidance)

#### 追踪阶段制导:
- 基于**视线角制导** (Line-of-Sight Guidance)
- 保持恒定纵向速度接近目标
- 横向控制用于对准

#### 对接阶段制导:
- 基于**相对位置误差**的制导
- 逐渐减小相对速度至零
- 精确控制对接姿态

## 3. Results (实验结果)

### 3.1 仿真设置
- **仿真平台**: MATLAB/Simulink
- **主动UAV**: 四旋翼模型
- **被动UAV**: 悬停或匀速移动
- **干扰**: 加入风扰等外部干扰

### 3.2 主要结果

#### 场景1: 与悬停UAV对接
- **结果**: 成功完成对接
- **对接精度**: 位置误差 < 5cm
- **收敛时间**: 约10-15秒

#### 场景2: 与移动UAV对接 (存在干扰)
- **结果**: 在干扰条件下成功对接
- **自适应能力**: 估计器能有效跟踪目标速度和干扰
- **鲁棒性**: 对风扰等干扰具有抑制能力

### 3.3 方法优势
1. **三阶段策略**: 逻辑清晰，便于实现
2. **自适应能力**: 无需精确知道目标速度
3. **抗干扰**: 能估计并补偿外部干扰
4. **通用性**: 适用于悬停和移动目标

---

## 4. 方法总结

### 核心创新点:
1. **分阶段对接策略**: 接近→追踪→对接，降低复杂度
2. **自适应估计**: 实时估计目标速度和干扰，无需先验知识
3. **反步控制**: 系统化的非线性控制器设计方法

### 适用场景:
- 四旋翼无人机空中对接
- 移动平台精准降落
- 多无人机协同组装

### 局限与改进方向:
- 仅考虑四旋翼，未涉及固定翼
- 视觉导航依赖标记，可考虑无标记方案
- 可进一步研究多机同时对接

## 5. 对你项目的启发

### 直接应用:
- **三阶段策略**可直接用于你的大四轴接小固定翼项目
- **自适应估计**可解决目标速度未知问题
- **反步控制**框架可用于非线性控制器设计

### 需要调整:
- 固定翼与四旋翼动力学不同，需修改模型
- 相对速度更大，制导律需重新设计
- 可考虑增加视觉伺服环节
